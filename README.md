# Tic Tac Toe Game

## URL: [Tic Tac Toe API](https://tttapi.vercel.app)

### Usage:

1. Clone the repository:

   ```bash
   git clone https://github.com/brianstm/tic-tac-toe.git
   ```

2. Install the required dependencies:

   ```bash
    npm install
   ```

3. Run the API Locally:
   ```bash
   npm start
   ```
